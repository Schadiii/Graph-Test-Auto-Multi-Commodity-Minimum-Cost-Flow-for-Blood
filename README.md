# Graph-Test-Auto-Multi-Commodity-Minimum-Cost-Flow-for-Blood

## Mathematical Modeling Final Course Output |BSM CS 3A

## Project Overview

This study develops a Multi-Commodity Minimum-Cost Flow model to optimize the distribution of blood from blood banks to hospitals in Malolos City, Bulacan. The system is represented as a directed graph, where healthcare facilities are modeled as nodes and transportation routes as edges with associated travel-time costs and capacity constraints.

## How to use

run 'blood_distribution.py' using your desired compiler.
